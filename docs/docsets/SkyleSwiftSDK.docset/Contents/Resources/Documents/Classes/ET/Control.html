Control

public class Control : ObservableObject

Control exposes Publisher which hold information about Skyles settings. These are partially user specific and should be pulled after a user is selected via Profile.select, Profiles.set or Profiles.delete.

  • enableStandby

    The enableStandby property exposes a Publisher which enables / disables the automatic standby mode. This mode allows Skyle to go into a standby mode when the iPad’s sceen is locked by pressing the power button for example. Skyle in standby should save some power and let the iPad live longer on battery power.
